# Budget Helper
To run, clone repo to your local machine. Run `npm install` to install dependencies. Then run `npm run dev` and navigate to the provided localhost in your favorite browser to see the website.

You can check to see if npm (Node Package Manager) is installed with npm --version. If it's not installed, you can navigate to [https://nodejs.org/] (https://nodejs.org/) and use the install instructions. If you're on mac and already have homebrew installed, you can just run `brew install node` to easily install Node.js & npm.
